# Music Data Analysis using Python & SQL
This project analyzes a music dataset using Python and SQL to extract insights about ratings, genres, emotions, languages, and artists.

## Tools Used:
- Python
- SQL
- Pandas
- Matplotlib
- Seaborn

## Key Highlights:
- Exploratory Data Analysis (EDA) on music data
- SQL- based data analysis
- Data visualization using Python graphs
- Trend analysis of songs, ratings, and genres
